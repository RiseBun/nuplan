## 🔧 Phase 1: 数据准备

### Step 1: 生成训练数据

使用 DrivingWorld 生成正负样本对：

```bash
cd /mnt/cpfs/prediction/lipeinan/nuplan

python generate_critic_training_data.py \
  --data-root /mnt/cpfs/prediction/lipeinan/nuplan_data/mini_set \
  --output-dir /mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data \
  --world-model-path DrivingWorld/pretrained_models/world_model.pth \
  --vqvae-path DrivingWorld/pretrained_models/video_vqvae.pth \
  --num-scenes 500 \
  --samples-per-scene 5 \
  --device cuda \
  --history-frames 4 \
  --future-frames 8
```

**参数说明**:
- `--num-scenes`: 场景数量（500 场景 ≈ 2500 样本）
- `--samples-per-scene`: 每个场景的样本数（1 正 + 4 负）
- `--history-frames`: 历史帧数（4 帧）
- `--future-frames`: 未来帧数（8 帧）

**预计时间**:
- 500 场景: ~4-6 小时（A100）
- 输出: ~50 GB

**输出**:
```
/mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data/
├── scene_00000_sample_0.pt  # 正样本
├── scene_00000_sample_1.pt  # 负样本
├── scene_00000_sample_2.pt  # 负样本
├── ...
└── generation_metadata.json
```

---

### Step 2: 计算多维度标签

为每个样本计算 6 个维度的标签：

```bash
python compute_training_labels.py \
  --data-dir /mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data \
  --output-dir /mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data_labeled \
  --device cuda \
  --num-samples 1000  # 可选：限制样本数
```

**计算的标签**:
1. `consistency_label` - 基于 FID（阈值 < 50）
2. `validity_label` - 综合判断
3. `speed_consistency_label` - 速度一致性
4. `steering_consistency_label` - 转向一致性
5. `progress_consistency_label` - 进度一致性
6. `temporal_coherence_label` - 时序连贯性

**预计时间**:
- 1000 样本: ~30-60 分钟

**输出**:
```
/mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data_labeled/
├── scene_00000_sample_0.pt  # 带标签
├── ...
└── labeling_metadata.json
```

---

### Step 3: 构建训练索引

生成 critic_train.jsonl 和 critic_val.jsonl：

```bash
python build_critic_index.py \
  --data-dir /mnt/cpfs/prediction/lipeinan/nuplan_data/critic_training_data_labeled \
  --output-dir /mnt/cpfs/prediction/lipeinan/nuplan/indices \
```

**参数说明**:
- `--train-ratio`: 训练集比例（0.8 = 80% 训练，20% 验证）
- `--balance-classes`: 平衡正负样本（推荐）
- `--seed`: 随机种子

**输出**:
```
/mnt/cpfs/prediction/lipeinan/nuplan/indices/

├── consistency_val.jsonl
└── index_metadata.json
```

```python
nuplan_scores = [...]  # 从 nuPlan Simulator 获取
```
