### 3. 相关性分析

```python
import json
import numpy as np
from scipy.stats import spearmanr, pearsonr

# 加载评估结果
with open('evaluation_results/summary.json') as f:
    results = json.load(f)

# 提取指标
fid_scores = [r['layer1_generation_quality']['fid'] for r in results]
critic_scores = [r['layer2_action_consistency']['consistency'] for r in results]

# 假设你有 nuPlan 闭环分数
nuplan_scores = [...]  # 从 nuPlan simulator 获取

# 计算相关性
fid_corr, fid_p = spearmanr(fid_scores, nuplan_scores)
critic_corr, critic_p = spearmanr(critic_scores, nuplan_scores)

print(f"FID vs nuPlan:     ρ = {fid_corr:.3f} (p = {fid_p:.4f})")
print(f"Critic vs nuPlan:  ρ = {critic_corr:.3f} (p = {critic_p:.4f})")

# 核心结论
if abs(critic_corr) > abs(fid_corr):
    print("\n✅ Critic 与 nuPlan 的相关性 > FID 与 nuPlan 的相关性")
    print("   证明 proposed metric 更有效！")
```

## 📊 指标解释

### Layer 1: 生成质量

| 指标 | 含义 | 范围 | 好坏 | 典型值 |
|------|------|------|------|--------|
| **FID** | 图像分布距离 | [0, ∞) | 越低越好 | < 50 |
| **FVD** | 视频时空距离 | [0, ∞) | 越低越好 | < 300 |
| **LPIPS** | 感知相似度 | [0, 1] | 越低越好 | < 0.2 |

**FID 参考**:
- 0: 完全相同的分布
- < 20: 优秀
- 20-50: 良好
- 50-100: 一般
- > 100: 较差

---

### Layer 2: Action 一致性

| 指标 | 含义 | 范围 | 好坏 | 典型值 |
|------|------|------|------|--------|
| **consistency** | 总体一致性 | [0, 1] | 越高越好 | > 0.75 |
| **speed_consistency** | 速度一致性 | [0, 1] | 越高越好 | > 0.70 |
| **steering_consistency** | 转向一致性 | [0, 1] | 越高越好 | > 0.70 |
| **progress_consistency** | 进度一致性 | [0, 1] | 越高越好 | > 0.65 |
| **temporal_coherence** | 时序连贯性 | [0, 1] | 越高越好 | > 0.75 |
| **validity** | 驾驶合理性 | [0, 1] | 越高越好 | > 0.80 |

---

### Layer 3: 驾驶合理性

| 指标 | 含义 | 范围 | 好坏 | 典型值 |
|------|------|------|------|--------|
| **NDCG@3** | Top-3 排序质量 | [0, 1] | 越高越好 | > 0.70 |
| **NDCG@5** | Top-5 排序质量 | [0, 1] | 越高越好 | > 0.75 |
| **MRR** | 平均倒数排名 | [0, 1] | 越高越好 | > 0.60 |
| **Top-1 Hit Rate** | 最佳命中率 | [0, 1] | 越高越好 | > 0.50 |
