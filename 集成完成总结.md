### Abstract

```
We propose a three-layer evaluation framework for driving world models:
- Layer 1: Generation quality (FID/FVD/LPIPS)
- Layer 2: Action consistency (multi-dimensional critic)
- Layer 3: Driving rationality (ranking metrics)

Our key finding: the proposed visual-action metric correlates better with 
nuPlan closed-loop performance (ρ=0.65) than traditional FID (ρ=0.42) 
or FVD (ρ=0.38).
```

### Introduction

```
1. Motivation: Current metrics (FID/FVD) don't correlate well with planning
2. Proposal: Three-layer evaluation framework
3. Key insight: Action consistency is crucial for planning
4. Contribution:
   - Complete evaluation framework
   - Multi-dimensional consistency critic
   - Empirical validation on nuPlan
```
