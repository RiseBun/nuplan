```python
class ClosedLoopMetricsComputer:
    """nuPlan 闭环指标（启发式模拟）"""
    
    def compute_no_at_fault_collision(trajectory, ground_truth):
        # NC: 无责任碰撞率 [0, 1]
        return score
    
    def compute_time_to_collision(trajectory, ground_truth):
        # TTC: 碰撞时间（秒）
        return ttc
    
    def compute_comfort(trajectory):
        # Comfort: 舒适度 [0, 1]
        return score
    
    def compute_progress(trajectory, ground_truth):
        # Progress: 进度 [0, 1]
        return score
    
    def compute_all_metrics(trajectory, ground_truth):
        # 综合分数
        overall = 0.4*NC + 0.2*(TTC/10) + 0.2*Comfort + 0.2*Progress
        return {NC, TTC, Comfort, Progress, Overall}
```

**功能**:
- ✅ 5 个闭环指标
- ✅ 基于轨迹偏离度、平滑度计算
- ✅ 加权综合分数

**注意**: 当前使用启发式指标模拟，后续可替换为真实 nuPlan simulator

```python
class CorrelationAnalyzer:
    """Spearman/Kendall 相关性分析"""
    
    def analyze_correlation(critic_scores, fid_scores, fvd_scores, closed_loop_metrics):
        # 计算 Spearman 相关性
        spearman_critic = spearman(critic_scores, closed_loop)
        spearman_fid = spearman(fid_scores, closed_loop)
        spearman_fvd = spearman(fvd_scores, closed_loop)
        
        # 计算 Gain
        gain_vs_fid = spearman_critic - spearman_fid
        gain_vs_fvd = spearman_critic - spearman_fvd
```

## 🎓 实验设计

### 实验流程

```
For each scene (100 scenes):
  1. 生成 10 个候选轨迹（噪声 0.0-0.5）
  
  2. 三种评分
     ├─ Critic 评分（越高越好）
     ├─ FID 评分（越低越好）
     └─ FVD 评分（越低越好）
  
  3. 选择 top-1
     ├─ Best by Critic
     ├─ Best by FID
     └─ Best by FVD
  
  4. 计算闭环指标
     ├─ NC (无责任碰撞率)
     ├─ TTC (碰撞时间)
     ├─ Comfort (舒适度)
     ├─ Progress (进度)
     └─ Overall (综合)

After all scenes:
  5. 计算 Spearman 相关性
     ├─ Critic vs 闭环指标
     ├─ FID vs 闭环指标
     └─ FVD vs 闭环指标
  
  6. 计算 Gain
     ├─ Gain vs FID
     └─ Gain vs FVD
  
  7. 可视化
```
