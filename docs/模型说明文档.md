# Consistency Critic 模型使用指南

## 📋 模型概述

**Action-Conditioned Future Consequence Evaluator** - 评估规划轨迹与视觉场景的一致性

**核心功能**: 给定当前图像和 ego action，评估候选轨迹的合理性

---

## 🔧 模型输入输出

### 输入格式

```python
{
    'history_images': [B, 4, 3, 224, 224],   # 4 帧历史图像
    'future_images':  [B, 4, 3, 224, 224],   # 4 帧未来图像（用于评估）
    'ego_state':      [B, 5],                # 自车状态
    'candidate_traj': [B, 8, 3]              # 候选轨迹
}
```

**字段说明**:
- `history_images`: 过去 4 帧前视相机图像
- `future_images`: 未来 4 帧图像（与 candidate_traj 对应）
- `ego_state`: `[x, y, heading, velocity, acceleration]`
- `candidate_traj`: 8 步轨迹，每步 `[dx, dy, dheading]`

---

### 输出格式

```python
{
    'consistency_logit':         [B],  # 整体一致性
    'validity_logit':            [B],  # 驾驶有效性
    'speed_consistency_logit':   [B],  # 速度一致性
    'steering_consistency_logit': [B], # 转向一致性
    'progress_consistency_logit': [B], # 进度一致性
    'temporal_coherence_logit':  [B],  # 时序连贯性
}
```

**使用方法**:
```python
# 转换为概率分数
consistency_score = torch.sigmoid(output['consistency_logit'])
validity_score = torch.sigmoid(output['validity_logit'])
# ... 其他维度

# 综合分数
overall = (
    1.0 * consistency_score +
    0.5 * validity_score +
    0.3 * speed_score +
    0.3 * steering_score +
    0.2 * progress_score +
    0.2 * temporal_score
)
```

## 🚀 使用方法

### 方法 1: 单条轨迹评估

```python
def evaluate_trajectory(model, scene, trajectory):
    """
    评估单条轨迹质量
    
    Args:
        scene: 包含 history_images, future_images, ego_state
        trajectory: [8, 3] 候选轨迹
    
    Returns:
        metrics: 多维度评分
    """
    model.eval()
    
    with torch.no_grad():
        batch = {
            'history_images': scene['history_images'].unsqueeze(0).cuda(),
            'future_images': scene['future_images'].unsqueeze(0).cuda(),
            'ego_state': scene['ego_state'].unsqueeze(0).cuda(),
            'candidate_traj': trajectory.unsqueeze(0).cuda(),
        }
        
        output = model(**batch)
        
        metrics = {
            'consistency': torch.sigmoid(output['consistency_logit']).item(),
            'validity': torch.sigmoid(output['validity_logit']).item(),
            'speed_consistency': torch.sigmoid(output['speed_consistency_logit']).item(),
            'steering_consistency': torch.sigmoid(output['steering_consistency_logit']).item(),
            'progress_consistency': torch.sigmoid(output['progress_consistency_logit']).item(),
            'temporal_coherence': torch.sigmoid(output['temporal_coherence_logit']).item(),
        }
        
        metrics['overall'] = (
            1.0 * metrics['consistency'] +
            0.5 * metrics['validity'] +
            0.3 * metrics['speed_consistency'] +
            0.3 * metrics['steering_consistency'] +
            0.2 * metrics['progress_consistency'] +
            0.2 * metrics['temporal_coherence']
        )
    
    return metrics

# 使用示例
metrics = evaluate_trajectory(model, scene, trajectory)
print(f"Overall Score: {metrics['overall']:.3f}")
```

### 方法 2: 多条轨迹排序

```python
def rank_candidates(model, scene, candidates):
    """
    对多个候选轨迹排序
    
    Args:
        candidates: List of [8, 3] 轨迹
    
    Returns:
        ranked_indices: 排序后的索引（高分到低分）
        scores: 每个候选的分数
    """
    model.eval()
    scores = []
    
    with torch.no_grad():
        for traj in candidates:
            batch = {
                'history_images': scene['history_images'].unsqueeze(0).cuda(),
                'future_images': scene['future_images'].unsqueeze(0).cuda(),
                'ego_state': scene['ego_state'].unsqueeze(0).cuda(),
                'candidate_traj': traj.unsqueeze(0).cuda(),
            }
            
            output = model(**batch)
            
            # 综合分数
            consistency = torch.sigmoid(output['consistency_logit']).item()
            validity = torch.sigmoid(output['validity_logit']).item()
            temporal = torch.sigmoid(output['temporal_coherence_logit']).item()
            
            score = 1.0 * consistency + 0.5 * validity + 0.3 * temporal
            scores.append(score)
    
    # 排序
    ranked_indices = sorted(range(len(scores)), 
                          key=lambda i: scores[i], 
                          reverse=True)
    
    return ranked_indices, scores

# 使用示例
candidates = [traj1, traj2, ..., traj10]  # 10 个候选
ranked_indices, scores = rank_candidates(model, scene, candidates)

print("排序结果:")
for i, idx in enumerate(ranked_indices[:3]):  # Top 3
    print(f"  Rank {i+1}: Score {scores[idx]:.3f}")
```

### 方法 3: 轨迹筛选

```python
def filter_candidates(model, scene, candidates, threshold=0.6):
    """
    筛选高质量轨迹
    
    Args:
        threshold: 分数阈值
    
    Returns:
        filtered: [(trajectory, score), ...]
    """
    ranked_indices, scores = rank_candidates(model, scene, candidates)
    
    # 筛选
    filtered = [(candidates[i], scores[i]) 
                for i in ranked_indices 
                if scores[i] >= threshold]
    
    return filtered

# 使用示例
filtered = filter_candidates(model, scene, candidates, threshold=0.6)
print(f"筛选后保留 {len(filtered)} 个候选")
```

## 🔗 闭环使用方法

### 闭环流程

```
1. 规划器生成多个候选轨迹
        ↓
2. Critic 评估每个候选的一致性分数
        ↓
3. 选择分数最高的轨迹
        ↓
4. 执行选择的轨迹
        ↓
5. 收集新的观测，重复步骤 1-4
```

---

### 完整闭环示例

```python
class CriticGuidedPlanner:
    """Critic 引导的规划器"""
    
    def __init__(self, critic_model, planner):
        """
        Args:
            critic_model: ConsistencyCriticModel
            planner: 基础规划器（生成候选轨迹）
        """
        self.critic = critic_model
        self.planner = planner
        self.critic.eval()
    
    def plan(self, scene, num_candidates=10):
        """
        闭环规划
        
        Args:
            scene: 当前场景
                {
                    'history_images': [4, 3, 224, 224],
                    'future_images': [4, 3, 224, 224],  # 可以用占位符
                    'ego_state': [5],
                }
            num_candidates: 候选轨迹数量
        
        Returns:
            best_trajectory: 最优轨迹 [8, 3]
            critic_score: Critic 分数
        """
        # Step 1: 生成候选轨迹
        candidates = self.planner.generate_candidates(
            scene, num_candidates
        )
        
        # Step 2: Critic 评估
        scores = []
        with torch.no_grad():
            for traj in candidates:
                batch = {
                    'history_images': scene['history_images'].unsqueeze(0).cuda(),
                    'future_images': scene['future_images'].unsqueeze(0).cuda(),
                    'ego_state': scene['ego_state'].unsqueeze(0).cuda(),
                    'candidate_traj': traj.unsqueeze(0).cuda(),
                }
                
                output = self.critic(**batch)
                
                # 综合分数
                consistency = torch.sigmoid(output['consistency_logit']).item()
                validity = torch.sigmoid(output['validity_logit']).item()
                temporal = torch.sigmoid(output['temporal_coherence_logit']).item()
                
                score = 1.0 * consistency + 0.5 * validity + 0.3 * temporal
                scores.append(score)
        
        # Step 3: 选择最优
        best_idx = scores.index(max(scores))
        best_trajectory = candidates[best_idx]
        
        return best_trajectory, scores[best_idx]

# 使用示例
critic_guided_planner = CriticGuidedPlanner(critic_model, base_planner)

# 闭环运行
for step in range(num_steps):
    # 获取当前观测
    scene = get_current_scene()
    
    # Critic 引导规划
    trajectory, score = critic_guided_planner.plan(scene)
    
    # 执行
    execute_trajectory(trajectory)
    
    print(f"Step {step}: Critic Score = {score:.3f}")
```

### 与传统方法对比

| 方法 | 流程 | 优缺点 |
|------|------|--------|
| **传统规划器** | 观测 → 规划 → 执行 | ❌ 不考虑视觉一致性 |
| **Critic 引导** | 观测 → 生成候选 → Critic 评分 → 选择最优 → 执行 | ✅ 考虑视觉-动作一致性 |

## 💡 实用技巧

### 1. 批量推理加速

```python
# 批量评估多个轨迹
def batch_evaluate(model, scene, candidates, batch_size=16):
    """批量推理"""
    model.eval()
    all_scores = []
    
    with torch.no_grad():
        for i in range(0, len(candidates), batch_size):
            batch_candidates = candidates[i:i+batch_size]
            
            # 构建 batch
            batch = {
                'history_images': scene['history_images'].unsqueeze(0).expand(len(batch_candidates), -1, -1, -1, -1).cuda(),
                'future_images': scene['future_images'].unsqueeze(0).expand(len(batch_candidates), -1, -1, -1, -1).cuda(),
                'ego_state': scene['ego_state'].unsqueeze(0).expand(len(batch_candidates), -1).cuda(),
                'candidate_traj': torch.stack(batch_candidates).cuda(),
            }
            
            output = model(**batch)
            
            consistency = torch.sigmoid(output['consistency_logit']).cpu().tolist()
            validity = torch.sigmoid(output['validity_logit']).cpu().tolist()
            temporal = torch.sigmoid(output['temporal_coherence_logit']).cpu().tolist()
            
            scores = [1.0*c + 0.5*v + 0.3*t 
                     for c, v, t in zip(consistency, validity, temporal)]
            all_scores.extend(scores)
    
    return all_scores
```

def safe_evaluate(model, scene, trajectory):
    """安全评估"""
    try:
        # 检查输入
        assert scene['history_images'].shape == (4, 3, 224, 224)
        assert scene['candidate_traj'].shape == (8, 3)
        
        # 评估
        metrics = evaluate_trajectory(model, scene, trajectory)
        return metrics
    
    except Exception as e:
        print(f"评估失败: {e}")
        return None
