```python
"""
nuPlan Closed-Loop 评估实验

目标: 证明 Critic 分数与闭环性能的相关性 > FID/FVD
"""

def experiment_closed_loop_correlation():
    """
    实验流程:
    
    1. 加载训练好的 Critic 模型
    2. 对每个场景生成多个候选轨迹
    3. 用 Critic 评分
    4. 在 nuPlan closed-loop 中运行
    5. 收集闭环指标
    6. 计算相关性
    """
    
    # Step 1: 准备测试场景
    scenes = load_nuplan_scenes(num_scenes=100)
    
    # Step 2: 生成候选轨迹
    for scene in scenes:
        candidates = generate_candidates(scene, num_candidates=10)
        
        # Step 3: Critic 评分
        critic_scores = []
        for traj in candidates:
            score = critic.evaluate(
                images=scene.history_images,
                actions=traj
            )
            critic_scores.append(score)
        
        # Step 4: FID/FVD 评分（对比基线）
        fid_scores = compute_fid(candidates, scene.ground_truth)
        
        # Step 5: 选择 top-1 轨迹
        best_by_critic = candidates[argmax(critic_scores)]
        best_by_fid = candidates[argmin(fid_scores)]
        
        # Step 6: Closed-loop 运行
        critic_performance = run_closed_loop(scene, best_by_critic)
        fid_performance = run_closed_loop(scene, best_by_fid)
        
        # Step 7: 记录结果
        results.append({
            'scene_id': scene.id,
            'critic_score': max(critic_scores),
            'fid_score': min(fid_scores),
            'critic_closed_loop': critic_performance,  # NC, TTC, etc.
            'fid_closed_loop': fid_performance,
        })
    
    # Step 8: 计算相关性
    spearman_critic = spearman_correlation(
        [r['critic_score'] for r in results],
        [r['critic_closed_loop']['NC'] for r in results]
    )
    
    spearman_fid = spearman_correlation(
        [r['fid_score'] for r in results],
        [r['fid_closed_loop']['NC'] for r in results]
    )
    
    print(f"Critic Spearman: {spearman_critic:.3f}")
    print(f"FID Spearman: {spearman_fid:.3f}")
    print(f"Gain: {spearman_critic - spearman_fid:.3f}")
```

#### 形式 B: 前向 consequence 一致性

> "evaluator 学到一个'合理 future consequence'空间，然后比较 generated future 与这个 consequence space 的一致性"

**状态**: ✅ **已完全实现**

**当前实现**:

我们的 **Consistency Critic** 就是形式 B 的实现：

```python
class MultiDimensionalConsistencyCritic(nn.Module):
    """
    学习"合理 future consequence"空间
    
    输入: 当前图像 + ego action
    输出: 6 维度一致性评分
    """
    
    def __init__(self):
        # 视觉编码器
        self.vision_encoder = resnet50(pretrained=True)
        
        # 动作编码器
        self.action_encoder = MLP(6 → 256)
        
        # 融合层
        self.fusion = TransformerEncoder()
        
        # 6 个输出头
        self.heads = {
            'consistency': Linear(512, 2),
            'speed': Linear(512, 2),
            'steering': Linear(512, 2),
            'progress': Linear(512, 2),
            'temporal': Linear(512, 1),
            'validity': Linear(512, 2),
        }
    
    def forward(self, images, actions):
        # 提取视觉特征
        vision_feat = self.vision_encoder(images)
        
        # 提取动作特征
        action_feat = self.action_encoder(actions)
        
        # 融合
        fused = self.fusion(vision_feat, action_feat)
        
        # 多维度评分
        return {head: module(fused) for head, module in self.heads.items()}
```

**关键设计**（符合需求）:

✅ **不是"像 expert 图像"**
```python
# ❌ 错误理解（像 GT）

# ✅ 正确理解（符合 action-conditioned manifold）

# 学习的是: "给定这个 action，这样的 future 是否合理"
```

✅ **学习 action-conditioned future consequence manifold**

```python
正样本: (images, real_action) → label=1  # 合理的 consequence

# 模型学到的是:
# "在 nuPlan 驾驶数据分布下，给定当前图像和 ego action，
#  合理未来图像应该长成什么样的统计规律"
```
