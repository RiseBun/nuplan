## 💡 关键创新点

### 1. 多维度标签生成

不是简单的二分类，而是 6 个维度的细粒度评估：

```python
labels = {
    'consistency_label': 1,           # 基于 FID
    'validity_label': 1,              # 综合判断
    'speed_consistency_label': 1,     # 速度一致性
    'steering_consistency_label': 1,  # 转向一致性
    'progress_consistency_label': 1,  # 进度一致性
    'temporal_coherence_label': 1,    # 时序连贯性
}
```

### 2. 正负样本生成策略

```python
# 正样本：使用真实轨迹
generated_good = world_model.generate(history, ego_state, real_trajectory)

# 负样本：使用扰动的轨迹
noisy_trajectory = real_trajectory + noise
generated_bad = world_model.generate(history, ego_state, noisy_trajectory)
```

### 3. 自动化训练流程

```
nuPlan 数据 
  → DrivingWorld 生成
  → 自动计算标签
  → 自动构建索引
  → 训练模型
  → 评估性能
```

全自动化，无需手动标注！
